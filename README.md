# Better Joiner
A mod for Mimesis that enhances the save system, room browsing, and provides an improved UI for game management.

## Features

### Infinite Saves
**Save Management**
- Browse unlimited save slots (no more slot limits)
- Improved save slot display with cycle count and timestamps
- View all save data at once with pagination
- Shows player names and save dates for quick identification
- Create new saves with automatic slot assignment

### Room Explorer
**Room Discovery**
- Browse available multiplayer rooms in real-time
- View room status and player count
- Auto-refresh room list every 5 seconds
- Filter rooms by available slots
- Shows dungeon name and hosting player

**Room Joining**
- One-click room joining
- Direct integration with Steam lobbies
- Automatic room list updates
- Handles room full/destroyed gracefully

### Enhanced UI
**Visual Improvements**
- Modern dark theme with professional styling
- Tabbed interface for saves and rooms
- Smooth pagination for browsing saves/rooms
- Clear status indicators and feedback
- Responsive design

**Save Slot UI**
- View 4 saves per page
- Sort by most recently modified
- Display save cycle, date, and player count
- Easy navigation with Prev/Next buttons
- Create new save button in new game mode

## Features Breakdown

### Saves Tab
- **Sort by Date** - Most recent saves shown first
- **Player Info** - See which players are in each save
- **Create New** - Add unlimited new save slots
- **Pagination** - Navigate through all saves with Prev/Next

### Rooms Tab
- **Auto-Search** - Continuously searches for available rooms
- **Real-time Updates** - Refreshes every 5 seconds
- **Player Count** - Shows current players vs max
- **Status Info** - Displays room status (Waiting, etc.)
- **Easy Join** - Single click to join selected room

## Hotkeys
- Close UI - ESC key (standard)
